Sequencia basica pra GIT
1  - git bash na pasta = abrir prompt git
2  - git init = para inciar codigo
3  - git status = saber se o codigo está atualizado
4  - git add "__.txt" = adicionar arquivo
5  - git add . = adicionar todos os arquivos
6  - git commit -m = adicionar versoes ou mudanças no codigo (boas praticas tbm)
7  - git remote add origin link do repositorio = definir repositorio inicial
8  - git push = subir arquivos do PC para servidor hub
9  - git branch __ = adicionar nova branch
10 - git branch = saber quantas branchs tem
11 - git checkout __ = mudar a branch utilizavel 
12 - git push --set-upstream origin staging = atualizar branch nova
13 - git pull = puxar ultima versão do codigo do serv pra maquina
14 - git merge __ = unir branchs para a master
15 - git checkout -b __ = criar e ja mudar para branch nova



Ciclo de Atualização
0 - git pull = puxar ultima versão do serv pra maquina
1 - git add . = adicionar todos arquivos e mudanças
2 - git commit -m "nome da mudança" = nomear versão do codigo
3 - git push = subir arquivo para o servidor
4 - git push --set-upstream origin staging = atualizar branch nova
5 - git merge = unir branchs para master
